Workshop / Demo

ExWBC: robust execution for long-horizon whole-body tasks.

The workshop focuses on execution-time feedback, recovery, progress estimation, memory, and robust whole-body control. It positions long-horizon execution as the missing link between learned skills and reliable real-world autonomy.

Live demonstration program

At ExWBC, invited presenters will demonstrate embodied systems that translate workshop themes into real-time humanoid execution.
