Track 1: OpenLane Topology

We propose OpenLane-v2 to shed light on the task of road structure perception and reasoning. The primary task of the dataset is road structure perception and reasoning, which requires the model to recognize the dynamic drivable states of lanes in the surrounding environment. The challenge of this dataset includes not only detecting lane centerlines and traffic elements but also recognizing the attribute of traffic elements and topology relationships on detected objects.

Detailed information about OpenLane-V2 can also be found on the Github

To evaluate performances on different aspects of the task, several metrics are adopted:
1. $mAP_{LC}$ for mAP on directed lane centerlines
2. $mAP_{TE}$ for mAP on traffic elements
3. $mAP_{LCLC}$ for mAP on topology among lane centerlines
4. $mAP_{LCTE}$ for mAP on topology between lane centerlines and traffic elements.
